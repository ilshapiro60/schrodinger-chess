# Schrödinger Chess

A unique 3D chess variant where strategy flows vertically between parallel boards.

## Game Rules

- **Boards**: 3 stacked 8x8 chess boards, each with a full set of pieces.
- **Moves**: Standard chess moves, but only on the same board.
- **Vertical Shifts**: After each move, the player selects one piece (except kings) to move vertically to an adjacent board if the target square is empty.
- **Objective**: Checkmate the opponent's kings. The player who checkmates more kings wins.

## Features

- Drag-and-drop piece movement
- Visual representation of 3 boards
- Built with Flutter for iOS and Android

## How to Run

1. Ensure Flutter is installed.
2. Connect a device or start an emulator.
3. Run `flutter run` in the project directory.
